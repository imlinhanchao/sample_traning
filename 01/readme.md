# 第一期项目

## [[后端] 3/21 猜数字](./backend)

猜数字是一个古老的游戏，由一个出题者随机出一个数字，由猜题者猜，若所猜数字大于所出数字，则出题者回答『大了』；若小于，则回答『小了』；直到猜题者猜中。现在请编写程序模拟出题者，随机数字在1~1000之间。当使用者猜中时，同时打印出猜测次数。

## [[爬虫] 3/21 获取IP](./crawler)

访问网站：ip.sxisa.com 可以得到本机的外网IP。现在，请编写程序，打印出本机的外网IP。
