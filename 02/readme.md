## [[后端] 4/11 保存猜数字成绩](./backend)

修改上一个程序，添加如下功能：  
1. 当用户猜到数字之后，提示其输入一个昵称作为游戏名称，然后保存成绩，包含昵称，开始时间，耗时，所猜数字，猜测次数；  
2. 游戏开启时，若用户已有设置了昵称，则输出提示语：你好！昵称~；  
3. 然后提示选择 1. 玩猜数字；2. 查看历史成绩；3. 更改设置昵称；  
4. 实现2，3功能。
